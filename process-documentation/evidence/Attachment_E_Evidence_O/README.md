|o|deploy their software in a container (i.e., utilized operating-system-level virtualization) |The team choose to deploy our software into an Amazon Web Service EC2 instance for production running the Docker daemon. It also provides a fast and efficient way to deploy our application in a dynamic and responsive fashion. Our Docker container running on the Docker hypervisor is our operating system-level virtualization. |
